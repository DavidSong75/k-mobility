# AI 주니어랩 6기 프로그램 일정표

카카오모빌리티 AI 주니어랩 6기의 3박 4일 프로그램과 담당 자료 폴더를 한 화면에서 확인하는 웹 시간표입니다.

## 공개 서비스

[https://davidsong75.github.io/k-mobility/](https://davidsong75.github.io/k-mobility/)

각 프로그램과 왼쪽 자료 메뉴를 선택하면 연결된 Google Drive 폴더가 새 창에서 열립니다.

## 로컬 실행

```bash
pnpm install
pnpm dev
```

## 프로덕션 빌드

```bash
pnpm build
```

`main` 브랜치에 변경사항을 푸시하면 GitHub Actions가 GitHub Pages에 자동 배포합니다.
